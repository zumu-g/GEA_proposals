# GEA Presentation Style Guide
## Modern Minimalist Designer Style

Based on analysis of the Nyora 2016 presentation template.

---

## 1. SLIDE DIMENSIONS

- **Size:** 26.67" x 15.00" (widescreen, double standard)
- **Aspect Ratio:** 16:9 (1.78:1)
- **Purpose:** Creates cinematic, impactful presentations

---

## 2. TYPOGRAPHY

### Headlines
- **Style:** Lowercase, no title case
- **Examples:** "discussion paper", "the problem", "our approach"
- **Merged words:** Creative compound headers like "refocuspriorities", "marketevidence"
- **Position:** Top-left corner, typically at (1.0", 1.0")
- **Size:** 60-100pt for main headlines
- **Weight:** Light or regular (not always bold)

### Subheadings
- **Style:** Short phrases, lowercase preferred
- **Size:** 24-36pt
- **Position:** Below headline or right side of slide

### Body Text
- **Style:** Sentence case, conversational
- **Size:** 18-24pt
- **Line spacing:** Generous (1.5-2.0)
- **Position:** Right side of slide (8-9" from left) or below headline
- **Max lines:** 5-7 per content block

### Font Choices
- **Primary:** Clean sans-serif (Helvetica Neue, Arial, Avenir)
- **Accent:** Modern serif for quotes (Georgia, Playfair Display)

---

## 3. LAYOUT PRINCIPLES

### The "Split" Layout
```
┌─────────────────────────────────────────┐
│                                         │
│  headline              • bullet point   │
│                        • bullet point   │
│                        • bullet point   │
│                                         │
│                                         │
└─────────────────────────────────────────┘
```
- Headline anchored top-left
- Content on right (starting ~8" from left)
- Bottom 30% often left empty

### The "Hero" Layout
```
┌─────────────────────────────────────────┐
│                                         │
│                                         │
│         single powerful                 │
│            statement                    │
│                                         │
│                                         │
└─────────────────────────────────────────┘
```
- Centered or left-aligned
- One idea, maximum impact
- Used for section dividers

### The "Image Left" Layout
```
┌─────────────────────────────────────────┐
│ ┌──────────┐                            │
│ │          │   headline                 │
│ │  IMAGE   │                            │
│ │          │   • content                │
│ │          │   • content                │
│ └──────────┘                            │
└─────────────────────────────────────────┘
```
- Image takes ~40% of left side
- Full height bleed
- Content on right with generous margin

### The "Grid" Layout
```
┌─────────────────────────────────────────┐
│  headline                               │
│                                         │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐    │
│  │     │  │     │  │     │  │     │    │
│  │  1  │  │  2  │  │  3  │  │  4  │    │
│  │     │  │     │  │     │  │     │    │
│  └─────┘  └─────┘  └─────┘  └─────┘    │
└─────────────────────────────────────────┘
```
- 2-4 items maximum
- Equal sizing
- Generous gaps between items

## 4. COLOR PALETTE

### Primary Colors
| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| Charcoal | #1A1A1A | 26, 26, 26 | Primary backgrounds |
| Off-White | #FAFAFA | 250, 250, 250 | Light backgrounds |
| Pure White | #FFFFFF | 255, 255, 255 | Text on dark |

### Accent Colors
| Name | Hex | RGB | Usage |
|------|-----|-----|-------|
| Warm Gold | #C4A962 | 196, 169, 98 | Highlights, accents |
| Sage Green | #8B9F82 | 139, 159, 130 | Secondary accent |
| Forest | #2D3830 | 45, 56, 48 | Dark accent blocks |

### Usage Rules
- **Dark slides:** Charcoal background, white text, gold accents
- **Light slides:** Off-white background, charcoal text
- **Never:** More than 3 colors per slide
- **Accent color:** Used sparingly for emphasis only

---

## 5. WHITESPACE

### Margins
- **Outer margin:** Minimum 1" on all sides
- **Preferred:** 1.5" left margin for headlines

### Breathing Room
- **Between headline and content:** Minimum 1.5"
- **Between content blocks:** Minimum 0.8"
- **Below last content:** Leave 25-30% of slide empty

### The "Less is More" Rule
- If slide feels balanced, remove one element
- Whitespace is a design element, not empty space
- Every element needs room to breathe

---

## 6. IMAGERY

### Style
- **Photography:** High-quality, editorial style
- **Mood:** Aspirational, lifestyle-focused
- **Treatment:** Full-bleed preferred

### Sizing
- **Full-bleed:** Image spans entire slide
- **Half-bleed:** Image spans full height, 40-50% width
- **Inset:** Smaller images with generous padding

### Placement
- Images typically on LEFT side
- Content overlays or sits to the right
- Never center-aligned images

---

## 7. SPECIAL ELEMENTS

### Section Dividers
- Simple, bold statement
- Centered or left-aligned
- Examples: "ends here", "the next step", "your investment"
- Dark background, white or gold text

### Accent Bars
- Thin horizontal lines (2-4pt)
- Gold color
- Used sparingly at top/bottom of slides

### Quote Blocks
- Italic serif font
- Centered
- Gold quotation marks or border accent

---

## 8. SLIDE TYPES

### Type 1: Title Slide
- Property address large and bold
- Minimal other elements
- Full-bleed image or solid dark background

### Type 2: Section Divider
- Single word or short phrase
- Lowercase
- Centered or bottom-left
- Dark background

### Type 3: Content Slide (Split)
- Headline top-left
- Content right side
- Generous whitespace

### Type 4: Image Feature
- Full or half-bleed image
- Minimal text overlay
- Let image do the talking

### Type 5: Data/Stats
- Large numbers
- Minimal labels
- Grid layout preferred

### Type 6: Closing
- Call to action
- Contact details subtle
- Confident, not salesy

---

## 9. DO'S AND DON'TS

### DO
- ✓ Use lowercase headlines
- ✓ Leave empty space
- ✓ One idea per slide
- ✓ Use full-bleed images
- ✓ Keep text minimal
- ✓ Use consistent positioning
- ✓ Create visual rhythm

### DON'T
- ✗ Center everything
- ✗ Fill every space
- ✗ Use bullet points excessively
- ✗ Mix too many fonts
- ✗ Use clip art or icons
- ✗ Crowd the slide
- ✗ Use title case for headlines

---

## 10. RHYTHM & FLOW

### Slide Sequence Pattern
1. **Dark slide** (statement/divider)
2. **Light slide** (content)
3. **Image slide** (visual break)
4. **Dark slide** (statement)
5. Repeat

### Pacing
- Section dividers between major topics
- Visual breathing room every 3-4 slides
- End sections with a pause slide
